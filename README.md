# youcompany course generator

Erste technische Grundlage für den Kursgenerator von youcompany.

## Ziel von Version 1

Der Kursgenerator soll aus einem Unternehmens-Text automatisch eine einfache Kursstruktur erzeugen.

## Output in Version 1

- Kurstitel
- 3 bis 5 Module
- je Modul 2 bis 4 Lektionen
- pro Lektion ein Textblock
- pro Lektion eine Quizfrage

## Noch nicht in Version 1

- PDF-Upload
- Mehrsprachigkeit
- Rollen- und Rechtesystem
- fertiges Design
- komplexe Bearbeitungslogik
