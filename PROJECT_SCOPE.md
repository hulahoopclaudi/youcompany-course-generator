# Project Scope

## Problem
Unternehmen haben Wissen, Prozesse und Standards, aber daraus schnell einen einfachen Kurs zu machen ist aufwendig.

## Ziel
Aus einem freien Text soll automatisch ein einfacher Minikurs entstehen.

## Zielgruppe Version 1
Unternehmen oder Admins von youcompany.

## Input Version 1
- Freitext über Job, Aufgabe, Prozess oder internes Wissen

## Output Version 1
- Kurstitel
- Kurzbeschreibung
- 3 bis 5 Module
- je Modul 2 bis 4 Lektionen
- pro Lektion:
  - Überschrift
  - Lerntext
  - eine Quizfrage
  - 3 Antwortoptionen
  - Markierung der richtigen Antwort

## Nicht in Version 1
- PDF-Upload
- Dateiverarbeitung
- Mehrsprachigkeit
- Nutzerrollen
- ausgefeilte UI
- Veröffentlichungsworkflow
- Analytics
- Zertifikate
- Bildgenerierung

## Erfolg für Version 1
Ein Text kann eingefügt werden und das System erzeugt daraus eine brauchbare erste Kursstruktur als JSON oder auf einer einfachen Seite.
